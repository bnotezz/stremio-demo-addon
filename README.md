# Blender Studio Static Stremio Addon

Статичний Stremio-сумісний аддон, який надає структурований каталог із популярними відкритими фільмами від студії **Blender Foundation** (з офіційного файлу `blender.movies.json`) з повними метаданими (IMDb/TMDb ID, постери, бекдропи, описи та стріми).

Оскільки аддон є повністю статичним, його можна хостити безкоштовно на будь-якому статичному хостингу (GitHub Pages, Netlify, Vercel тощо).

## ✨ Особливості

- **Інтеграція з IMDb/TMDB:** Використовуються офіційні ідентифікатори IMDb (`tt...`) або TMDb (`tmdb-...`), що дозволяє Stremio-сумісним плеєрам автоматично пов'язувати ваші стріми з глобальною базою даних.
- **Повні метадані:** Додано якісні постери, фонові зображення (fanart), тривалість, оцінки, та описи безпосередньо з бази TMDb.
- **Різноманітні формати стрімів:** Підтримка нативних `ytId` для YouTube-відео, `infoHash` для торрентів, а також класичних прямих MP4/WEBM посилань.

## 🚀 Встановлення

Встановіть аддон у ваш Stremio-сумісний плеєр за допомогою посилання на маніфест (замініть `YOUR_HOSTED_URL` на URL вашого хостингу):

```text
https://YOUR_HOSTED_URL/manifest.json
```

1. Скопіюйте посилання на маніфест.
2. Відкрийте ваш плеєр.
3. Перейдіть до розділу **Addons** (Аддони) ➔ **Install from URL** (або вставте посилання в пошуковий рядок аддонів).
4. Підтвердьте встановлення.

## 🛠️ Локальна розробка та збірка

### 1. Встановлення залежностей
```bash
npm install
```

### 2. Збірка статичних JSON файлів
Для генерації файлів аддону в директорію `dist/stremio` виконайте:
```bash
npm run build
```

Ця команда зчитає `blender.movies.json`, скомпілює генератор `src/generate-stremio.ts` та запустить його. Сгенерована структура папок міститиме:
- `manifest.json` — маніфест аддону.
- `catalog/movie/blender_studio_catalog.json` — список фільмів для каталогу.
- `meta/movie/*.json` — детальні метадані для кожного фільму.
- `stream/movie/*.json` — доступні потоки відтворення.

### 3. Перевірка типів
```bash
npm run typecheck
```
